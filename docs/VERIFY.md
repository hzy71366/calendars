## 验证步骤

### 方法一：AirDrop 传输（推荐）

1. 将 `docs/liuzhai.ics` 通过 AirDrop 发送到你的 iPhone / Mac
2. 接收后点击文件 → 选择「日历」打开
3. 系统自动导入，显示为「六斋日」日历

### 方法二：文件传输

1. 将 `docs/liuzhai.ics` 通过 iCloud、微信、邮件等方式传输到 iPhone
2. 用「文件」App 打开 → 选择「日历」导入

### 方法三：macOS 直接导入

```bash
# 在 Mac 终端中，双击打开或运行：
open docs/liuzhai.ics
# 或手动拖入日历 App
```

## 如果遇到问题

| 现象 | 原因 | 解决 |
|:----|:-----|:-----|
| 日历名称显示 `null` | 订阅方式不支持 `X-WR-CALNAME` | 改用导入 `.ics` 文件方式 |
| 标题 emoji 显示方框 | iOS 版本过旧 | 升级至 iOS 16+ |
| 日期不对 | 时区转换问题 | 全天事件不需要时区，确认 DTSTART 是 `VALUE=DATE` 格式 |
| 导入后无反应 | 文件关联没设置 | 用 AirDrop 或 macOS 日历直接打开 |
